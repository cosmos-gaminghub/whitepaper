# Cosmos Gaming Hub
Cosmos Gaming Hub whitepaper documents.

## Develop Environment
- node
- gitbook
- yarn or npm

## Get Started

**install gitbook** with yarn

```
$ yarn global add gitbook-cli --prefix /usr/local
```

or npm

```
$ npm install --global gitbook-cli
```

**start gitbook**

```
$ git clone https://github.com/cosmos-gaminghub/whitepaper.git
$ cd whitepaper
$ gitbook serve
```

see localhost:4000
