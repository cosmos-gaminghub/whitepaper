# GAME Whitepaper

GAME whitepaper documents.


## Develop Environment
- node
- HonKit

## Get Started


npm

```
$ npm install
```

**start HonKit

```
$ git clone https://github.com/cosmos-gaminghub/whitepaper.git
$ cd whitepaper
$ npx honkit serve
```

see http://localhost:4000
